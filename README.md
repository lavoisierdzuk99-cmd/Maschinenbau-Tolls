# Engineering Toolkit

Dieses Tool bietet interaktive Rechner für grundlegende maschinenbauliche Aufgaben:

- Schrauben: Vorspannkraft, Sicherheitsabschätzung
- Biegung & Torsion: Standardfälle mit sichtbaren Formeln
- Welle & Lager: Auslegung mit Vergleichsspannung
- ISO-Passung (H/h): Spiel, Abmaße, Passungsart
- Werkstoffe: Vergleich wichtiger Kennwerte + Kurzkommentare
- Kostenkalkulation: einfache Teilekalkulation
- Wärmeübertragung: 1D-Wand, U-Wert, Wärmestrom

Hinweis: Alle Rechnungen sind didaktische Vereinfachungen
und ersetzen keine vollständigen Norm- und Festigkeitsnachweise.
